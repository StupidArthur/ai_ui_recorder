# 测试用例：用户登录并进入工作台

## 1. 业务背景与初始状态
用户访问登录页面，需要使用手机号和密码完成身份认证，同意用户协议后登录系统。登录成功后自动跳转至工作台主界面，显示用户信息及导航菜单。

## 2. 测试步骤流

### [步骤 1] 输入用户名
- **执行动作**：在「请输入用户名」文本框中输入手机号
- **状态验证**：输入框内容由空变为「15700078644」，表单状态已更新

### [步骤 2] 输入密码
- **执行动作**：在「请输入密码」密码框中输入密码
- **状态验证**：密码框内容由空变为掩码形式（•••••••••••），表单状态已更新

### [步骤 3] 点击协议勾选区域
- **执行动作**：点击协议勾选区域
- **状态验证**：无可见变化

### [步骤 4] 勾选用户协议
- **执行动作**：在同意协议复选框中点击勾选
- **状态验证**：复选框从未选中状态变为已选中状态

### [步骤 5] 提交登录表单
- **执行动作**：点击「立即登录」按钮
- **状态验证**：登录表单消失，页面跳转到工作台界面，显示「我的对话」「我的」「Agent」「我的应用」「我的数据」「我的模型」等导航项，以及用户信息「余泽超」「14000」

# 测试用例：用户通过菜单修改偏好设置并关闭面板

## 1. 业务背景与初始状态
测试用户通过点击头像打开用户菜单，进入偏好设置面板进行主题和显示开关的配置操作，验证修改后关闭面板的功能完整性。

## 2. 测试步骤流

### [步骤 6] 打开用户菜单
- **执行动作**：点击用户头像图片
- **状态验证**：页面左侧滑出用户菜单面板，显示「余泽超」「黄金会员」「偏好设置」「系统管理」「反馈记录」「通知公告」「退出登录」等选项

### [步骤 7] 进入偏好设置面板
- **执行动作**：点击用户菜单中的「偏好设置」选项
- **状态验证**：弹出偏好设置面板，显示外观（浅色/深色切换）、模式（简洁/常规/详细模式）、字体大小滑块（当前值20）、首页显示内容开关（案例介绍、推荐问题）、Agent显示所有对话过程开关、「恢复默认设置」按钮

### [步骤 8] 切换显示模式
- **执行动作**：点击模式选项区域中的某项
- **状态验证**：模式选择状态发生变化，「详细模式」不再标记为「使用中」状态

### [步骤 9] 开启对话过程显示开关
- **执行动作**：点击「显示所有对话过程」开关按钮
- **状态验证**：该开关从未选中状态变为已选中状态

### [步骤 10] 选择推荐问题
- **执行动作**：点击推荐问题列表中的某项
- **状态验证**：聊天输入框内容更新为新的推荐问题，界面出现Agent入口区域（工业Agent、各类Agent名称列表），「显示所有对话过程」开关变为选中状态

### [步骤 11] 尝试选择深色主题
- **执行动作**：点击「深色」主题选项
- **状态验证**：无可见变化

### [步骤 12] 确认切换深色主题
- **执行动作**：点击深色主题单选按钮
- **状态验证**：浅色主题单选按钮取消选中，深色主题单选按钮变为选中状态，界面恢复为初始推荐问题列表

### [步骤 13] 关闭偏好设置面板
- **执行动作**：点击「Close」关闭按钮
- **状态验证**：偏好设置面板关闭，聊天输入框恢复为空，推荐问题区域重新显示

# 测试用例：用户偏好设置重置与恢复

## 1. 业务背景与初始状态
用户已进入系统主界面，当前主题为深色模式，各偏好开关保持选中状态。用户通过点击头像区域进入用户菜单，计划进行偏好设置的相关操作。

## 2. 测试步骤流

### [步骤 14] 展开用户菜单
- **执行动作**：点击用户头像图片区域
- **状态验证**：页面右侧弹出用户菜单，显示用户名「余泽超」、会员等级「黄金会员」及「偏好设置」「系统管理」「反馈记录」「通知公告」「退出登录」等菜单项

### [步骤 15] 进入偏好设置面板
- **执行动作**：点击「偏好设置」菜单项
- **状态验证**：用户菜单关闭，偏好设置面板重新展开，当前主题为深色模式，各开关保持选中状态

### [步骤 16] 恢复默认设置
- **执行动作**：点击「恢复默认设置」按钮
- **状态验证**：偏好设置重置：主题由深色切换为浅色，多个开关项（案例介绍、推荐问题等）由选中状态变为取消选中，推荐问题列表内容已刷新，用户菜单区域关闭

### [步骤 17] 更新推荐问题列表
- **执行动作**：点击推荐问题列表区域
- **状态验证**：推荐问题列表内容更新，偏好设置面板中「详细模式」标签移至「使用中」右侧

### [步骤 18] 开启工业Agent显示开关
- **执行动作**：点击偏好设置面板中第3个开关按钮
- **状态验证**：该开关由取消选中变为选中状态，首页显示内容恢复显示工业Agent入口、查看更多及多个Agent卡片（阀门异常检测-通用、离子膜预测Agent、AI PID整定 Agent等）

### [步骤 19] 关闭偏好设置面板
- **执行动作**：点击「关闭」按钮关闭偏好设置面板
- **状态验证**：偏好设置面板已关闭，页面恢复至主界面状态

# 测试用例：进入专家模式并向智能问数专家提问

## 1. 业务背景与初始状态
用户在工业数据智能助手的主界面中，需通过专家模式获取专业领域的问答服务。本次验证从进入专家模式、选择智能问数专家、提出问题并获得解答、到对回答进行点赞反馈的完整交互流程。

## 2. 测试步骤流

### [步骤 20] 进入专家模式
- **执行动作**：点击「专家模式」标签
- **状态验证**：页面新增显示多个AI专家卡片，包括AI操作优化专家、AI复杂控制专家、AI回路整定专家、AI生产参数预测专家、预测控制专家、回路整定专家、能耗瓶颈分析专家、AI换热网络优化专家、工业知识专家、报表分析专家、智能问数专家等

### [步骤 21] 选择智能问数与结果分析专家
- **执行动作**：点击「智能问数与结果分析」专家卡片
- **状态验证**：页面切换至智能问数专家对话界面，输入框显示「输入内容进行提问」，推荐问题列表更新为新的问题内容（分离冷箱冻堵风险、冷箱通道压降增大、急冷水塔塔盘堵塞、裂解炉在线清焦周期缩短等），专家模式AI专家卡片列表已收起

### [步骤 22] 输入提问内容
- **执行动作**：在输入框中输入「有哪些可以问」
- **状态验证**：输入框内容已更新为「有哪些可以问」

### [步骤 23] 提交问题并获取AI回答
- **执行动作**：点击发送按钮提交问题
- **状态验证**：对话列表新增「有哪些可以问」提问条目，AI助手返回工业数据智能助手功能说明，包含能耗物耗指标查询、设备位号数据查询、系统与规则咨询三类内容

### [步骤 24] 对回答进行点赞反馈
- **执行动作**：点击「点赞」按钮
- **状态验证**：点赞图标更新为勾选圆圈，提示文字显示「感谢您的赞美与肯定」
